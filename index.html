    Local Video Player

    PreviousNext

    Win Forms application used as Netflix clone used for movies and tv shows saved locally.

    • IR Hub project connects via serial port to relay commands from regular TV remote and Arduino Remote project connects via TCP to relay mouse movement from joystick; displays on TV using laptop connected via HDMI
    • Requires VLC Media Player to be installed as LibVLCSharp is leveraged to play media so video can be rendered using laptop GPU
    • Uses The Movie Database (TMDB) API to download images (movie posters, episode backdrops, season posters, etc.) and media descriptions
    • Demonstration of custom made 128x128 cursor made for ease of viewing on TV where resolution is much larger than intended for regular sized cursor

    Arduino Remote and Infrared (IR) Hub

    PreviousNext

    Remote and IR Hub provides control over Local Video Player application, Samsung TV, and Bomaker soundbar.

    • Remote can control Samsung TV and Bomaker sound bar independently using IRremote library
    • IrScrutinizer version 2.3.0 used to reconstruct IR signals for discontinued sound bar with help of RemoteMaster file uploaded to hifi-remote.com
    • ESP8288 ESP-01 Wi-Fi module used to relay joystick movement to laptop for mouse control via TCP communication application (also integrated into Local Video Player project)
    • IR Hub features:
      • relay IR commands so unused buttons on Samsung TV remote can be used to control sound bar
      • connects to Local Video Player application via serial port to provide control via infrared
      • intercepts Samsung TV power signal and relays power on/off signal for sound bar
    • Version 1 of remote

    iCloud Notes

    PreviousNext

    Win Forms application which syncs basic note interface with iCloud Notes.

    • CEF Sharp library (embed Chromium in .NET app) used to manipulate DOM on www.icloud.com/notes
    • 3 variations:
      • master: Basic note taker which uses RTF file for saving locally
      • cef-sharp: Keeps browser page for iCloud Notes open for application duration
      • cef-sharp-cloud-save: Created due to high resource usage by previous variation
    • cef-sharp-cloud-save branch features:
      • To keep Chromium from hogging resources, the browser is used only twice at launch and exit of application to restore/save notes
      • Microsoft.Office.Interop.Word library used to preserve formatting between text save/load between Windows and Mac Application
      • Custom context menu item which allows time for notes to save before OS shutdown
      • Login credentials only required once using CEF Sharp cache
    1. C#
      100.0%

    Dell Hardware Monitor

    PreviousNext

    Win Forms application to monitor status of Dell G5 and Dell 7559 laptops.

    • Libre Hardware Monitor used monitor the temperature sensors, voltages, load and clock speeds
    • Uses DellFanManagement library to monitor and control fans on Dell laptops
    • Due to limitations of EC control only 2 speeds available for fan controls: low ~2000RPM and high ~4000RPM (see archived reference)
    • Low cpu <0.5% and memory usage ~20MB while polling for updates every 2 seconds
    • Demonstration of fan control

    Mini CNC Pen Plotter

    PreviousNext

    Computer numerical control (CNC) plotter moves its head with the instructions provided by software to draw using ballpoint pen.

    • Built with Arduino Uno using 3 stepper motors:
      1. 28BYJ-48 stepper motor with ULN2003 driver
      2. PowerMac G4 DVD Drive stepper motor with L293D driver
      3. USB Slim 2.0 External DVD Drive stepper motor with L293D driver
    • Uses g-code , which is code telling the motors what type of action to perform, produced from Inkscape and feeds input to Arduino through serial port using Processing
    • Demonstration of plotting: May 2019 , Squidward , and the Earth (regular speed)

    Obstacle Avoiding RC Car V2

    PreviousNext

    Arduino RC Car that uses Win Forms application to bridge communication between Wii remote and car.

    • Wiimote uses bluetooth Human Interface Device Profile (HID) and is not compatible with HC-05 module since it uses Serial Port Profile (SPP)
    • Can be manually control using Wii remote motion sensors: i.e. tilt the remote forward to accelerate, backwards to reverse, and left and right for turning
    • Wiimote buttons setup: A for accelerate, B for reverse, 1 switch to manual, &2 switch to auto
    • Added servo motor for precision control of front wheels and changed photoresistors from last version to tilt switch with barrier in front to detect obstacles lower than sensor view
    • Demonstration of car using manual control, switching to auto-control, then back to manual
    • Version 1 (no manual control)
